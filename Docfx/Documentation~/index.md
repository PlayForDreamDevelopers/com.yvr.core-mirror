# Docfx

`DocFx` 是微软用来生成 `MSDN` 文档的工具，可以自动从 `.Net` 的源码中生成 API Documentation，也可以通过 `markdown` 生成 Articles 。

本仓库将作为各第一方应用的 Submodule，以提供文档模板的功能。

本文档将最大程度的介绍日常使用中所依赖的 Docfx 的各种特性及遇到的问题，更多关于 Docfx 的资料可参考：
- [Docfx 官方文档](https://dotnet.github.io/docfx/index.html)
- [Docfx 开源仓库](https://github.com/dotnet/docfx)
- [Docfx 官方视频教程](https://learn.microsoft.com/en-us/shows/on-net/intro-to-docfx)
